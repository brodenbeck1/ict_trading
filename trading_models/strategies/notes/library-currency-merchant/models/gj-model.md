# The GJ Model (Preludes of Market Expansions)

The CM Academy module on **recognizing that an expansion is about to begin** — the
curriculum describes it as "preludes of market expansions (GJ model)." The name
likely derives from GBP/JPY ("GJ" in forex shorthand), the violently-expanding pair
Kishane's forex audience associates with the behavior — the model generalizes the
pre-expansion signature that pair exhibits constantly.

The publicly inferable shape of a prelude, consistent with the rest of his system:
contraction into a key time (tightening cycle ranges / inside conditions), a final
liquidity grab against the impending direction, and the first displacement away —
the cues that the PA flow is transitioning consolidation → expansion. The module's
purpose in the curriculum is sequencing: GJ identifies *that and when* expansion
begins; Type 1/Type 2 define *how* to board it.

**Details are paywalled.** Populate from source videos via

## Detection Rules (provisional scaffold)

- Prelude features, evaluated near key times: declining cycle range (e.g., 3
  successively smaller 90-min cycle heights), inside-bar accumulation, coiling around
  a session open, failed pokes both sides.
- Trigger: first displacement leg out of the contraction (ICT displacement detector)
  — emits `expansion_started` for the Type 1/2 entry models.
- Research framing: measure expansion-day precursors on ES/NQ/YM (overnight range
  percentile, Asia/London compression) to give this file real, instrument-specific
